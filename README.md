# Bacteroides across the gut
Repository for data and analysis code associated with:
"Sex and regional effects of Bacteroides in the gut"
Rebecca A. Valls, Kaitlyn E. Barrack, Sarvesh V. Surve, James B. Bliska, George A. O'Toole
Department of Microbiology and Immunology, Geisel School of Medicine at Dartmouth, Hanover, USA.
This repository contains data from experiments examining the effects of a Bacteroides cocktail across gut regions in male and female mice.

# Contents
16s_analysis/ — DADA2 pipeline, Phyloseq pipeline, ASV sequences, count tables, taxonomy assignments, and sample metadata for 16S rRNA microbiome sequencing

CFUs/ — Colony forming unit data and R analysis across gut segments and across experiment timeline

Luminex_analysis/ — Cytokine multiplex data from serum and lungs and R analysis

MiceWeight/ — Body weight tracking data and R analysis

NanoString/ — Raw RCC files with metadata file, normalized count data, and R analysis
